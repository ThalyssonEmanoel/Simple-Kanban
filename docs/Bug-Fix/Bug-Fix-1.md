# Bug-Fix 1
## Autenticação
- Problemas com login, ao logar o usuário não entra direto na tela de login, ele precisa atualizar a página, apertar F5, para ai sim entrar na tela de projetos,  a princípio isso só acontece no computador, devo investigar e solucionar esse problema.

## Interface do usuário
### Modal de login
- Não conseguem visualizar a própria senha, precisa adicionar um ícone de olho que permita visualizar a senha.
###Modal de criação de conta
- Não conseguem visualizar a própria senha, precisa adicionar um ícone de olho que permita visualizar a senha.
### Tela dentro do projeto

- Permitir que o líder, ou o próprio usuário que criou a atividade, possa adicionar multiplas pessoas;

coloquei que a data é 14/10/2026 ela fica como 13/10/2026.
### Tela de configurações do projeto
