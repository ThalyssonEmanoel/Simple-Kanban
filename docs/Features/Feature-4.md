# Adições de funcionalidades
- Adicionar a opção do proprietário excluir um projeto, com confirmação digitando o nome do projeto e exclusão em cascata de colunas, cartões, comentários, anexos e notificações;
- Adicionar a opção do usuário logado alterar a própria senha a partir do modal de perfil, informando a senha atual e a nova senha (com confirmação) e com ícone de olho para mostrar/ocultar cada campo;

# Correções/Modificações de funcionalidades
- Melhorar a estilização do modal de exclusão de projeto, com cabeçalho de alerta, botão de confirmação visível (cinza quando desabilitado, vermelho quando habilitado) e layout responsivo;
- Ajustar o empilhamento (z-index) dos modais para que o modal de alterar senha apareça sobre o modal de perfil.
